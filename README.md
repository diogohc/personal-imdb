# personal-imdb
This is a spring-boot application to build a IMDB clone like site. It's a personal project to improve my technical skills while working with something I like to do in my spare time, wathing movies.

## Tech Stack
 - Spring Boot
 - Java
 - Kotlin
 - RabbitMQ
 - MySQL
